# 有何不可設計｜Why Not Design

這是 `whynot-tw.github.io` 的靜態網站 repository。

## 網站定位

有何不可設計的公開網站入口，用於：

- 品牌介紹
- 服務項目
- 作品集整理
- Design OS 查詢入口
- 未來串接 Airtable / JSON / 靜態資料
